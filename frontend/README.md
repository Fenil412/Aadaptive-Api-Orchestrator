# Frontend Dashboard - RL API Orchestrator

This React application acts as a Dashboard for the RL API Orchestrator system, communicating with the FastAPI backend.

## Quickstart

1. Provide the Backend URL. In `.env`, you can set `VITE_API_BASE_URL=http://localhost:8001`.
2. Install Node packages:
   ```bash
   npm install
   ```
3. Run Development Server:
   ```bash
   npm run dev
   ```
